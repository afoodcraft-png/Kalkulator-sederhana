# Learn HTML — Mini Course

This small project provides a few lesson pages and an interactive editor to practice HTML.

How to view locally:

```bash
# start a simple HTTP server on port 8000
python3 -m http.server 8000

# then open http://localhost:8000/index.html in your browser
```

Files:

- [index.html](index.html) — main index
- [styles.css](styles.css) — simple styles
- [examples/basic-structure.html](examples/basic-structure.html) — lesson
- [examples/forms.html](examples/forms.html) — form example
- [examples/interactive.html](examples/interactive.html) — live editor

Try editing the example files and refresh the browser to learn.
